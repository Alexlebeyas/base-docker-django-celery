What is vagrant

Vagrant is a virtual environment manager. We use virtualbox to create our virtual environments with vagrant boxes. These boxes are operating system images pre-configured to work with vagrant. Vagrant set virtualbox settings like synced folders, forwarded ports, RAM, head and headless and much more.

Why use it

It's the best of both worlds. Use a nice Windows desktop and code in whatever OS you need. Mostly ubuntu 14.04. All of our production environments run on ubuntu 14.04, that is why we need to reproduce in development. Some development technologies like sqlite are not exactly the same as their production equivalient.
