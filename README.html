What is project starter

Project starter is a fully configured and blank django project with all of our tech. It is meant to be copied into a new repository and installed via the install command.

Create a new project

  1. Copy all the files from the /project_starter/ directory into your new git repository.
  2. Create a python3 virtualenv if you need to. The installation script runs tests and they will most likely fail without the right python version.
  3. Open a terminal at the new project directory and execute the install command with python3 and your new project name:

    >> python install my_new_project_name

  4. Read the output of the installation and make sure that migrations and tests of libs ran successfully. The install script will have deleted itself.

Add project variables

The install package found inside the /project_starter/ directory is well documented. It contains utilities and tasks functions. Tasks are functions that will (most of the time) search and replace placeholder texts to input dynamic data. You will find more comments and examples in /install/tasks.py.
