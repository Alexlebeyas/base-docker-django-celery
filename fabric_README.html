What is fabric

Fabric is a simple, Pythonic tool for remote execution and deployment. It manages most of our old projects. From your terminal, you can send ssh management commands to remote servers.

Commands

Fabric commands are self made and located in fabfile.py in the root of a project. The deployment function is pretty much the only one you need.

>> fab deploy

Easy to use

We don't use fabric anymore, but you can. It's much more simple than puppet, and in python. Read the docs!
